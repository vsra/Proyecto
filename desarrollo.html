¿Qué es un desarrollador web?

Un desarrollador web es aquel que codifica, diseña y lleva a cabo tareas lógicas, basándose en lo que el diseñador web popuso. El desarrollador es el que cuenta con el conocimiento en lenguajes de programación, desde la creación con la estructura básica en html, llegando a crear y mantener un servidor web, con el lenguaje correspondiente.

¿Qué es un diseñador web?

Un diseñador web diseña la pagina, aplicando directivas, se esfuerza en que la página sea intuitiva, amigable y accesible.

Diferencias entre desarrollador y diseñador web

El diseñador web, es aquel que diseña toda la web, encargándose de cada funcionalidad de ésta página, así como la usabilidad que ésta posea, los colores apropiados, así como el menú e imágenes y videos que ésta pueda contener. El diseñador, cuenta con conocimientos, principalmente en todos los programas de diseño, como lo son por ejemplo Photoshop , Ilustrator, DreamWeaver. El programador emplea todos sus conocimientos, para llevar a la realidad todo lo que el diseñador, diseñó.

¿Qué es HTML?

HTML es un lenguaje de etiquetas, dando significado a sus siglas como HyperText Markup Language, sirve para indicar cómo va ordenado el contenido de una página web. Esto lo hace por medio de las marcas de hipertexto las cuales son etiquetas conocidas en inglés como tags.

¿Qué es CSS?

CSS es un lenguaje de estilo que define la presentación de los documentos HTML. De siglas CascadingStyle Sheets. CSS abarca cuestiones relativas a fuentes, colores, márgenes, líneas, altura, anchura, imágenes de fondo, posicionamiento avanzado y muchos otros temas. CSS es utilizado para añadir formato al sitio web, ofreciendo más opciones a la hora de diseñar el mismo, además cuenta con un soporte amplio en distintas versiones de navegadores.

¿Qué es JavaScript?

JavaScript es un lenguaje de programación que se utiliza principalmente para crear páginas web dinámicas.Es un lenguaje de programación interpretado, por lo que no es necesario compilar los programas para ejecutarlos. En otras palabras, los programas escritos con JavaScript se pueden probar directamente en cualquier navegador sin necesidad de procesos intermedios.

¿Qué es Bootstrap?

Bootstrap es un framework desarrollado y liberado por Twitter que tiene como objetivo facilitar el diseño web. Permite crear de forma sencilla webs de diseño adaptable, es decir, que se ajusten a cualquier dispositivo y tamaño de pantalla y siempre se vean igual de bien. Es Open Source o código abierto, por lo que lo podemos usar de forma gratuita y sin restricciones.

¿Qué es JQuery?

JQuery es un framework Javascript, sirve como base para la programación avanzada de aplicaciones, aportando una serie de funciones o códigos para realizar tareas habituales. Permite simplificar la manera de interactuar con los documentos HTML, manejar eventos, desarrollar animaciones. Fue presentada el 14 de enero de 2006 en el BarCamp NYC. JQuery es la biblioteca de JavaScript más utilizada

¿Qué es JSon?

JSON (JavaScript Object Notation) es un formato para el intercambios de datos, básicamente JSON describe los datos con una sintaxis dedicada que se usa para identificar y gestionar los datos. JSON nació como una alternativa a XML, el fácil uso en javascript ha generado un gran numero de seguidores de esta alternativa. Una de las mayores ventajas que tiene el uso de JSON es que puede ser leído por cualquier lenguaje de programación. Por lo tanto, puede ser usado para el intercambio de información entre distintas tecnologías.
